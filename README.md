# Budgeting Tool

A personal finance web application designed to help users track budgets, transactions, and loan information in a single platform.

Built as a full-stack project using React, TypeScript, Python, and PostgreSQL.

---

## Project Status

**<ins>Early Development</ins>**

Current progress:
- Backend API initialized
- Database structure created
- User authentication implemented
- Login page functional (currently redirects to a placeholder dashboard)

Deployment has not yet been released.

---

## Planned Features

- Budget creation and management
- Excel transaction import
- Loan amortization tracking
- Spending visualization
- Budget sharing between users
- UI refinement

## Why I Built This

I built this project to deepen my understanding of full-stack architecture while creating a practical financial tool focused on budgeting, financial visibility, and long-term planning.

The goal is to design a clean, scalable system that combines secure authentication, structured financial data modeling, and intuitive user experience.

---

## Future Direction

Development is focused on expanding core budgeting functionality, improving security practices, and preparing the application for its first production deployment.
